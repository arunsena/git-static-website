Website on GitHub

How to host a static website using GitHub for free

The Internet is a world of web pages and there are millions of websites out there. It’s always an amazing way to portray yourself through a personal website or your business website which expands your horizons. When it comes to website hosting, there are many popular ways to do it whether it is getting a domain on GoDaddy, Namecheap, Hostgator or through site builders like Wix or Wordpress. Not to forget, Amazon Web Services (AWS), a cloud-based platform for hosting at a low cost per usage. Here, I am talking about GitHub — a popular developer portal to discover, share and deploy code. GitHub allows you to host static website quickly in simple steps over secure https protocol for free

Create a account in GitHub

Go to GitHub to create an account

Once you create an account— two ways to get your website in minutes If black screen so-called command prompt scares you, please skip to next step and make your life simple with GitHub Desktop app

Using Command line

Once, you log in to GitHub

  • Create a new repository using a plus icon on the top right corner of the page
  • Give a name to the repo, keep the default options and click ‘create repository’
  • Now, let’s hold on here and go to desktop and open command prompt (Enter ‘cmd’ in the start menu on Windows or search for ‘terminal’ on Mac)
  • Create a new directory using the command ‘mkdir’ in your command prompt and ‘cd’ to that folder where you want to add your code
  • Make sure you have git installed locally using ‘git — version’. If you don’t have one, please refer to this link on installation steps
  • Run ‘git init’ to initialize empty git repo in the folder you created
  • Add your HTML and other resources to this folder
  • Once you add your files, run ‘git add .’ command to stage your files
  • Now, run ‘git commit -m “initial commit”’ to commit your changes and run ‘git status’ to check the status
  • Let’s add the remote URL we got when we created the repo and push to publish your changes to the right repo in GitHub
  • Please enter GitHub Credentials if prompted
  • Go to your GitHub account and now you should see your files under your newly created repo
  • Now navigate to setting page for this repo on the top right corner and go to GitHub Pages
  • Select the source and change it to master and click save
  • Click the generated link in the GitHub pages section and you should see your webpage
  • Congratulations on getting your website!!

Using GitHub Desktop App

GitHub desktop is pretty simple to use, you can get one here — Github Desktop

  • Once you install the app, click to create a new repository
  • Give a name, description and a local path to your repo and click ‘Create Repository ’ button
  • Navigate to the folder in finder or windows explorer and add files
  • Come back to the app and you will see your files have been staged here
  • Now give a commit message in the bottom left box and commit changes
  • Once you commit your changes, click the ‘Publish Repository’ on the top
  • Please enter GitHub Credentials if prompted
  • Go to your GitHub account and now you should see your files under your newly created repo
  • Now navigate to setting page for this repo on the top right corner and go to GitHub Pages
  • Select the source and change it to master and click save
  • Click the generated link in the GitHub pages section and you should see your webpage
  • Congratulations on getting your website!!
